# Glossary

confidence: HIGH
source: QUESTION.md, BOARD.md, spiffe-provenance.md, the-question-session-2026-06-21/README.md

Core vocabulary for The Question. Use these terms; do not drift to synonyms.

- **Case** - one investigation. Board note holding symptom, timeline, status (Open/Closed). Closed cases persist.
- **Suspect** - one hypothesis. Falsifiable prediction with verdict (Untested -> Testing -> Confirmed/Ruled Out).
- **Source** - one piece of evidence with origin and confidence.
- **SVID** - Subject-Verifiable Investigation Document. The six-field provenance record on a board node. Reframe of SPIFFE's SVID; evidentiary, not cryptographic.
- **Trust domain** - the project/workspace; root of a spiffe:// ID.
- **Citation gate** - tiered rule: hard for findings (must be `verified`), soft for color (flagged).
- **Case zero** - the founding handwritten concept note (`docs/origins/the-question-original-concept.md`).
- **Faceless** - terse compressed response mode; re-voice of the retired caveman mode.
- **Function over costume** - if persona/board files are missing, skills still run and drop the flavor; the mask is never a hard dependency.
