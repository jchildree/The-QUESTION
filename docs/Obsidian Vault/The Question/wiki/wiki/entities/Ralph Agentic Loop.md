# Ralph Agentic Loop

Coding-agent pattern by Geoffrey Huntley (July 2025). A coding agent run in a bash loop: fresh context each iteration, state surviving on disk (TODO / git) rather than conversation memory, one task per pass.

## Relation to The Question

The case-zero note's "like Ralph?" aside resolved to this. Verdict: lineage / inspiration only. The Question borrows the philosophy -- fresh context, disk-as-memory, one-task discipline -- but is not a literal loop runner.
